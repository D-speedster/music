### 🔧 نصب و راه‌اندازی

#### 1. نصب وابستگی‌ها:
```bash
pip install -r requirements.txt
```

#### 2. پیکربندی:
```bash
cp .env.example .env
# ویرایش .env و افزودن API_ID, API_HASH, BOT_TOKEN
```

#### 3. اجرای ربات:
```bash
python music_bot.py
```

#### 4. تست اتصال:
```bash
python test_bot.py
```
